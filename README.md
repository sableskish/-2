## Общее описание: 
инструмент предназначен для анализа и визуализации зависимостей пакетов в репозиториях Alpine Linux. Программа позволяет получать информацию о зависимостях пакетов, строить граф зависимостей и создавать его графическое представление.
реализованы первые два этапа проекта:
1. минимальный прототип с конфигурацией
2. сбор данных о зависимостях
## Функции и настройки: 
### Функции:
1. анализ конфигурации - проверка корректности введенных параметров
2. получение зависимостей - извлечение информации о прямых зависимостях пакетов из репозитория Alpine Linux
3. валидация данных - обработка ошибок и проверка входных параметров
4. гибкая настройка - поддержка различных режимов работы и фильтрации
